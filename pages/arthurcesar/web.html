1 - Explique o que é CGI.

Em informática CGI (sigla em inglês para Common Gateway Interface, Interface Comum de Ligação) é uma interface padrão multilinguagem que permite servidores Web acessar as funcionalidades em um computador servidor para gerar páginas dinâmicas,permitindo a um navegador passar parâmetros através do script CGI para um programa executado em um servidor web e, assim gerar a página depois de os processar.

Explique o que é Servidor Web, Container Web (Java) e Servidor de Aplicação.

Web Server - Servidor Web - é uma software que conversa com browsers via HTTP e presa pela escalabilidade.

O Web Container é uma forma de se referir ao ambiente onde os Servlets rodam. É incluso um mecanismo de CGI e processamento de HTTP. A diferença entre Web Server e Web Container é que Web Server é generico para qualquer tecnologia e se refere a um produto, normalmente.

Servidor de Aplicação - é o resultado da especificação java EE. É um software que contém um Web Container, um EJB Container, acesso JNDI e JMS ambiente tansacional com JTA e muitas outras tecnologias como Java Mail , JDBC e CORBA que juntas formam o ambiente EE

Defina Arquitetura Cliente-Servidor.

A tecnologia cliente/servidor é uma arquitetura na qual o processamento da informação é dividido em módulos ou processos distintos. Um processo é responsável pela manutenção da informação (servidores) e outros responsáveis pela obtenção dos dados (os clientes)

Defina Arquitetura MVC. E fale das vantagens e desvantagens da utilização do MVC

Vantagens:

1 - Como o modelo MVC gerencia múltiplos visualizadores usando o mesmo modelo é fácil manter , testar e atualizar sistemas múltiplos

2 - É muito simples incluir novos clientes apenas incluindo seus visualizadores e controles

3 - Torna a aplicação escaláve

Desvantagens:

1 - Requer uma quantidade maior de tempo para analisar e modelar o sistema

2 - Requer pessoal especializado

3 - Não é aconselhável para pequenas aplicações
