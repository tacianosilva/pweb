O que é servidor web e o que é servidor de aplicação? Qual a diferença?

Servidor web é um computador que hospeda sites, e somente via esses computadores é fornecido o endereço de algum site. Um servidor de aplicação disponibiliza uma plataforma que separe do desenvolvedor algumas das complexidades de um sistema como segurança.
