O que é servidor web e o que é servidor de aplicação? Qual a diferença?

Um Servidor de Aplicação serve processos que permitem clientes fazerem requisições e receberem respostas. Já um Servidor Web é um servidor de aplicação especializado para atender as necessidades Web.Ou seja, precisa trabalhar com protocolos HTTP e derivados.

Pesquise e fale um pouco sobre branches e pull request usando git e github. Escreva um resumo.

Os branches permitem que você desenvolva recursos, corrija erros ou experimente com segurança novas ideias em uma área contida do seu repositório. Já As pull requests permitem que você informe outras pessoas sobre as alterações das quais você fez push para um branch em um repositório no GitHub. Depois que uma pull request é aberta, você pode discutir e revisar as possíveis alterações com colaboradores e adicionar commits de acompanhamento antes que as alterações sofram merge no branch base.
